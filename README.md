## 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/GAsecure/Insuranceadvisorgaurava.git
```

Open the project folder:

```bash
cd Insuranceadvisorgaurava
```

Run locally using any web server or simply open `index.html` in your browser.

---

## 🌍 GitHub Pages Deployment

1. Push all files to the `main` branch.
2. Open **Repository → Settings → Pages**.
3. Source:
   - **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
4. Save the settings.
5. Your website will be available at:

```
https://gasecure.github.io/Insuranceadvisorgaurava/
```
